# WALMART-_SALES-_FORECAST
This project focuses on forecasting retail sales for Walmart using historical sales data and statistical analysis. The objective is to understand sales behavior, identify patterns such as seasonality and trends, and build a reliable forecasting model that can support inventory planning and business decision-making.

Instructions to Run the Project

Download all files from this repository to your local system.

Open the Jupyter Notebook file provided in the project.

Inside the Jupyter Notebook, provide the correct file path to the dataset where required
(update the file path in the code according to the location where you downloaded the files).

Once the correct file paths are set, run the notebook cells, and the code will run properly.
